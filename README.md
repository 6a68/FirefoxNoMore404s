## Developing

This is a "WebExtension". See https://developer.mozilla.org/en-US/Add-ons/WebExtensions/Your_first_WebExtension

- Upgrade to Firefox 45 or newer
- Visit `about:debugging`
- Click "Load Temporary Add-on"
- Click "manifest.json"

## Building

- Run `make build`
